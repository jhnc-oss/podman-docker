# podman-docker

[![ci](https://github.com/jhnc-oss/podman-docker/actions/workflows/ci.yml/badge.svg)](https://github.com/jhnc-oss/podman-docker/actions/workflows/ci.yml)
![GitHub](https://img.shields.io/github/license/jhnc-oss/podman-docker?color=yellow)

Docker adapter for Podman. Unlike the official [Podman Docker emulation](https://github.com/containers/podman/blob/main/docker), some CLI arguments and formats are modified to match Docker.
